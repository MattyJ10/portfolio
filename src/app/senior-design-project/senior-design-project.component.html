Configurable Mesh Network

July 2018 - Present

About

This is my senior design project for Santa Clara University. My partner and I are creating a configurable mesh network that is especially useful in situations where networking infrastructure is not available. These scenarios occur during disaster recovery, in rural areas and more.

Responsibilities

  • Creating REST API with node.js and ExpressJS
  • Designing frontend application with Angular

Note

This project can be found on my GitHub. It is still currently in production and will be finalized by May 2019.

What I'm Doing

Designing the frontend with Angular to display network data in an organized manner. This enables the network admin to see the current topology of the network, connected devices, which devices are sending which data and more.

Building tools to allow configuration of the network through the frontend. This allows the network admin to set the route packets take, prioritize certain IPs, move a connected user to a different mesh node and more.

Created REST API to communicate with mesh nodes on the network. The nodes will then transfer files with network data to the server which will parse the data and send it back to the frontend.
